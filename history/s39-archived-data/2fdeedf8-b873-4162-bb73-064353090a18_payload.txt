 Got it! So the URL pattern is:

```
/viewprofile/{firstName}/{memberId}
```

For future reference, if you ever need to construct this URL manually instead of using the `link-members-3-name` field, it would be:

```javascript
to(`/viewprofile/${currentUser.name.toLowerCase()}/${currentUser._id}`)
```

Or if the name field might have spaces or special characters:

```javascript
to(`/viewprofile/${currentUser.name.toLowerCase().replace(/\s+/g, '-')}/${currentUser._id}`)
```

But using `currentUser["link-members-3-name"]` is the safest approach since Wix generates that dynamic page link automatically and handles all the formatting for you.
